# MSPlayer
### 基于AVPlayer的视频播放器
* 支持右滑前进、左滑后退
* 支持滑动滑竿来调节进度
* 支持滑动屏幕左侧调节亮度
* 支持滑动屏幕右侧调节音量
* 竖屏下支持横屏
* 支持自动横屏竖屏
* 支持本地和网络视频
